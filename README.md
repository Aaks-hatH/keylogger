# Educational Keylogger

A Python script demonstrating how to use the `pynput` library to hook into system keyboard events.

## Disclaimer
**For Educational Purposes Only.** This software is developed for learning concepts regarding system hooks and input monitoring. Using this software on systems without permission is illegal.

## Usage
1. Install dependencies: `pip install pynput`
2. Run script: `python main.py`
3. Check `key_log.txt` for output.
